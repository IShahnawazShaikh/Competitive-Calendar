# Competitive Calendar
 
‘Competitive Calendar’ is a coding Calendar developed for Coding enthusiastic who never want to miss their contest and looking for upcoming Contest on different platforms like Codeforces, CodeChef, Atcode, Leetcode, HackerRank and HackerEarth.
Features of this app are:
1.	‘Competitive Calendar’ allows you to share the coding contest detail through WhatsApp.
2. 	You can register for the contest through this app
3.	Contest details updated whenever u will swipe refresh through particular contest page.
4.	Contest status is available like upcoming, end and running.
